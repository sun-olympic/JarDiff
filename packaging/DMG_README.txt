JarDiff — JAR 包代码对比工具
================================

安装步骤
--------
1. 把本窗口里的「JarDiff」拖到右侧的「应用程序」文件夹。

2. 首次运行前，请打开「终端」执行下面这行命令（去掉系统隔离标记，
   否则可能提示「已损坏」或「无法验证开发者」）：

       xattr -dr com.apple.quarantine /Applications/JarDiff.app

   或者：在「应用程序」里右键点击 JarDiff →「打开」→ 在弹窗中再点「打开」。

3. 之后正常双击即可使用。


使用方法
--------
- 在顶部填写「旧版 JAR」「新版 JAR」，支持三种来源：
    本地路径、HTTP(S) 链接、Maven 坐标（groupId:artifactId:version）
- 选择反编译方式（默认 CFR，可把 .class 反编译成 Java 源码）。
- 点「开始对比」，左侧树形列出变更文件，点击文件即可在右侧并排查看代码差异。
- 「仓库设置」里可修改 Maven 仓库地址并保存。


说明
----
- 本应用已自包含，目标电脑无需安装 Python 或 Java。
- 反编译器 CFR 会在首次需要时自动下载，请确保能联网访问对应 Maven 仓库。
- 本版本未经过 Apple 公证，仅供内部使用。
